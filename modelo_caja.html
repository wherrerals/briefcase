Modelo de Caja CSS

Todos los elementos html son como cajas que tienen contenido, borde, relleno, y margen

el contenido podemos manipularlo y dale altura y un ancho

El padding es el espacio en tre el contenidoy el borde, podemos darle valores de arriba, abajo, izquierda y derecha

border es el borde que delimita nuestra caja

el margin es el espacio que se genera entre estas
